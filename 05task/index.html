Перебираючі методи масивів

Створіть функцію getRandomArray(length, min, max),
яка повертає масив випадкових цілих чисел.
У функції є параметри:
length - довжина масиву;
min – мінімальне значення цілого числа;
max – максимальне значення цілого числа.
Наприклад, getRandomArray(15, 1, 100) –> [6, 2, 55, 11, 78, 2, 55, 77, 57, 87, 23, 2, 56, 3, 2]
Створіть функцію getModa(...numbers),
яка вираховує моду всіх переданих в неї аргументів.
Нецілі числа ігноруються.
Наприклад, getModa(6, 2, 55, 11, 78, 2, 55, 77, 57, 87, 23, 2, 56, 3, 2) –> 2
Створіть функцію getAverage(...numbers),
яка рахує середнє арифметичне всіх переданих в неї аргументів.
Нецілі числа ігноруються.
Наприклад, getAverage(6, 2, 55, 11, 78, 2, 55, 77, 57, 87, 23, 2, 56, 3, 2) –> 34.4
Створіть функцію getMedian(...numbers),
яка рахує медіану всіх переданих в неї аргументів.
Нецілі числа ігноруються.
Наприклад, getMedian(6, 2, 55, 11, 78, 2, 55, 77, 57, 87, 23, 2, 56, 3, 2) –> 23;
getMedian(1, 2, 3, 4) –> 2.5;
getMedian(1, 2, 3, 4, 5) –> 3
Створіть функцію filterEvenNumbers(...numbers),
яка фільтрує парні числа передані як аргументи функції
Наприклад, filterEvenNumbers(1, 2, 3, 4, 5, 6) -> [1, 3, 5]
Створіть функцію countPositiveNumbers(...numbers),
яка порахує кількість чисел більших 0.
Наприклад, countPositiveNumbers(1, -2, 3, -4, -5, 6) -> 3
Створіть функцію getDividedByFive(...numbers),
яка відфільтрує усі елементи в масиві та залишить тільки ті,
які діляться на ціло на 5.
Наприклад, getDividedByFive(6, 2, 55, 11, 78, 2, 55, 77, 57, 87, 23, 2, 56, 3, 2) -> [55, 55]
Створіть функцію replaceBadWords(string),
яка 1) розіб'є фразу на слова,
2) замінить погані слова на зірочки (*). При вирішенні цього завдання необхідно розбити масив на слова за допомогою функції .split(" "),
після чого масив необхідно буде склеїти .join(" ") Погані слова: shit та fuck.
Передбачте можливість розширювати список цих слів у майбутньому.
Наприклад, replaceBadWords("Are you fucking kidding?") -> "Are you ****ing kidding?"
replaceBadWords("Holy shit!") -> "Holy ****!"
replaceBadWords("It's bullshit!") -> "It's bull****!"
Створіть функцію divideByThree(word),
яка розбиває кожне слово на умовні склади по 3 букви.
Якщо букв менше трьох – не розбиває.
Пробіли завжди видаляються. Рядок приводится до нижнього регістру.
Наприклад, divideByThree("Commander) -> ["com", "man", "der"]
divideByThree("live") -> ["liv", "e"]
Створіть функцію generateCombinations(word),
яка видасть всі можливі перестановки(унікальні, без повторень) букв в слові.
Для тестів не передавайте слова в яких більше 10 букв.
Краще взагалі обмежити работу функції 10 буквами.
Наприклад, generateCombinations("man") -> ["man", "mna", "amn", "anm", "nam", "nma"]
generateCombinations("ol") -> ["ol", "lo"]
